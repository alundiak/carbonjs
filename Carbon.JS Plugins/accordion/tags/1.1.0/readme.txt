Carbon.JS accordion-plugin

Синтаксис:
Q(elem).accordion({
	orientation: string,
	timeout: number,
	event: string,
	oneOpened: boolean,
	onOpening: function,
	onOpened: function,
	onClosing: function,
	onClosed: function
});

Описание:
Функция accordion() позволяет вставлять на страницу слайдер-"гармошку".
Для работы функции необходимо чтобы были подключены модули animation, dom и utilities.

Настройки для функции:
orientation - вертикальная или горизонтальная ориентация слайдера, соответственно, "v" или "h". По умолчанию "v".
timeout - время смены вкладок гармошки в миллисекундах. По умолчанию это значение равно 500 мс.
event - обработчик события, при котором будет активизироваться вкладка. По умолчанию "click".
oneOpened - булево значение, указывающее на то, могут ли в данный момент времени все вкладки быть закрытыми. Если true - всегда будет оставаться одна открытая вкладка. По умолчанию true.
onOpening - функция, которая будет выполняться циклически при открытии какой-либо вкладки. Внутри переданной функции this будет указывать на открывающуюся вкладку в обёртке Q().
onOpened - функция, которая выполнится после того как вкладка открылась. Внутри переданной функции this будет указывать на только что открытую вкладку в обёртке Q().
onClosing - функция, которая будет выполняться циклически при закрытии какой-либо вкладки. Внутри переданной функции this будет указывать на закрывающуюся вкладку в обёртке Q().
onClosed - функция, которая выполнится после того как вкладка закрылась. Внутри переданной функции this будет указывать на только что закрытую вкладку в обёртке Q().

Функция использует следующую структуру кода (пример на тегах <div>):
<div id="accordion">
	<div class="acc-header">Заголовок вкладки 1</div>
	<div class="acc-content">Содержимое вкладки 1</div>
	...
	<div class="acc-header">Заголовок вкладки N</div>
	<div class="acc-content">Содержимое вкладки N</div>
</div>

Это может быть и список <ul>, главное, чтобы был контейнер (к нему применяется функция), содержащий чередующиеся элементы с классами, начинающимися на acc-header и acc-content

Q("#accordion").accordion({...});

Установка:
Создайте папку plugins/accordion в папке с модулями Carbon.JS и закачайте в неё содержимое trunk одноимённой папки plugins/accordion из репозитория фреймворка. Затем необходимо вручную подключить файл carbonjs.plugin.accordion.packed.js
