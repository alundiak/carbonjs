Carbon.JS shadow-plugin

Синтаксис:
Q().shadow({
	to: string,
	color: string,
	opacity: number,
	method: string,
	time: number
});

Описание:
Функция устанавливает на странице тень (содержимое страницы затемняется).
Для работы функции необходимо чтобы были подключены модули animation и utilities.

Она принимает следующие настройки: 
to - строковый параметр, отвечающий за установку (show) и убирание тени (hide). По умолчанию show.
color - шестнадцатеричный код цвета тени, по умолчанию чёрный (#000000)
opacity - конечная прозрачность тени, по умолчанию 50
method - строковый параметр, соответствующий одному из методов CarbonJS.Transitions в виде строки (например, "Sine.EaseOut"), по умолчанию "Line.EaseIn"
time - время появления и исчезновения тени, по умолчанию 0 мс

Пример:
Q().shadow({
	time: 500
});
- поставить чёрную полупрозрачную тень за 500 мс.

Установка:
Создайте папку plugins/shadow в папке с модулями Carbon.JS и закачайте в неё содержимое trunk одноимённой папки plugins/shadow из репозитория фреймворка. Затем необходимо вручную подключить файл carbonjs.plugin.shadow.packed.js
